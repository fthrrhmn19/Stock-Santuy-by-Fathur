Metodologi analisa

Analisa memakai indikator apa?

Dashboard memadukan indikator teknikal, level harga, fair value proxy, risk scoring, dan timing.

Teknikal Utama

  • Trend: EMA 9, 20, 50, dan 200.
  • Momentum: RSI dan MACD.
  • Volume: relative volume dan volume spike.
  • Support, resistance, target, dan stop loss berbasis ATR.

Mode Analisa

  • Day trade memakai candle intraday saat tersedia.
  • Swing trade memakai struktur harian.
  • Long term memakai trend panjang dan stabilitas.
  • Semua mode punya skor dan alasan terpisah.

Batas Akurasi

  • API gratis IDX biasanya delayed atau tidak resmi.
  • Real-time resmi membutuhkan lisensi data IDX.
  • Gunakan hasil sebagai alat bantu, bukan instruksi transaksi.
  • Selalu cek orderbook broker sebelum eksekusi.
